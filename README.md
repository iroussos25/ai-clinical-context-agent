# Aegis AI: Clinical Decision Support (Next.js)

This is the Next.js implementation of the Aegis clinical AI platform. 
For the full project overview, architectural comparison, and documentation, 
see the [Aegis Project monorepo](https://github.com/iroussos25/aegis-project).

**[Live Demo](https://aegis-ai-cds.vercel.app/)**

## Quick start
```bash
cp .env.example .env.local
# Add Google AI API key and Supabase credentials
npm install
npm run dev
```

## Tech stack

Next.js 14, TypeScript, React 19, Vercel AI SDK, Supabase pgvector, 
Upstash Redis, Zod, PubMed API, HL7 FHIR R4, Tailwind CSS, 
Framer Motion, Vitest, GitHub Actions
